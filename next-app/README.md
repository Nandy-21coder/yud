# Vercel-Ready Next.js Application

This is a production-ready Next.js 14 application scaffolded with:
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Deployment on Vercel
This project is configured to be seamlessly deployed on Vercel. 
1. Import this project to your Vercel dashboard.
2. Vercel will automatically detect the Next.js framework.
3. Configure any environment variables specified in `.env.example`.
4. Deploy!

## Local Development
Run `npm run dev` to start the development server on `localhost:3000`.
